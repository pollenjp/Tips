# gnu Screen command

- Ctrl-a
    - Ctrl-c    : create new window
    - Ctrl-"    : show all windows
    - Ctrl-a    : toggle window
    - k         : kill the window
    
    - S         :(split)
                    split horizantally
    - Ctrl-|    :(split)
                    split vertically
    - Ctrl-Tab,
      Tab       :(focus)
                    Switch the input focus to the next region.
    - X    :(remove)
                    kill the current region.
